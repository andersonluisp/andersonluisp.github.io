    TodoApp

    App desenvolvido em parceria com Diego Ribeiro para auxiliar na gestão de atividades do dia a dia, possibilitando definir prioridades, prazo e recebimento de notificações. Entrei para auxiliar no desenvolvimento e implantar novas features como DateTimePicker, prazo para realização das tarefas, notificações e novo layout.
    Repositório Github

    Tecnologias utilizadas:

  • ROOM
  • POO
  • MVVM
  • DateTime Picker
  • Recyclerview
  • Coroutines
  • Fragments
  • Tratamento de exceptions
